# 🎬 Movie Recommender System

A Machine Learning-based Movie Recommender System that suggests movies similar to the one selected by the user. This project uses content-based filtering to recommend movies based on their features.

---

## 🚀 Features

- Recommend movies based on similarity
- Fast and interactive UI using Streamlit
- Clean and user-friendly interface
- Real-time recommendations

---

## 🧠 How It Works

This project uses **Content-Based Filtering**:
- Movies are recommended based on similarity scores
- Similarity is calculated using cosine similarity
- Features like genres, keywords, cast, and crew are used

---

## 🛠️ Tech Stack

- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Streamlit (for web app)
- Jupyter Notebook (for model building)
- PyCharm (for development)
